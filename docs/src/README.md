# wow-patcher

World of Warcraft client patcher for TrinityCore private servers.

## Overview

wow-patcher is a Rust-based tool for patching WoW game clients to work with
TrinityCore private servers. It handles binary modification and configuration
updates.

## Features

- Binary patching for WoW clients
- Version detection and compatibility
- Configuration management
- Safe patch operations with rollback support
